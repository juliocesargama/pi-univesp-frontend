Este projeto foi desenvolvido para o Projeto Integrador do curso de Tecnologia da Informação da Universidade Virtual do Estado de São Paulo (UNIVESP) para o segundo semestre no ano de 2023 no qual segue o tema:

Desenvolver um software com framework web que utilize banco de dados, inclua script web (Javascript), nuvem, uso de API, acessibilidade, controle de versão e testes. Opcionalmente, incluir análise de dados.

Elaborado a partir do framework Vue.js, o projeto consiste em um sistema de gerenciamento de empréstimos de itens para as escolas públicas do município de Piracaia.
